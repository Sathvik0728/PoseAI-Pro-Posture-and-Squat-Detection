# PoseAI Pro – Real-Time Body Posture & Squat Detection

## Overview

PoseAI Pro is an AI-powered fitness and posture detection application that performs real-time body posture tracking and squat counting using computer vision and MediaPipe Pose Estimation.

The project uses a webcam feed to detect body landmarks, analyze knee joint angles, and accurately count squats while displaying live posture analytics on screen.

## Features

- Real-time body posture detection
- AI-based squat counter
- Knee angle detection
- Live webcam pose tracking
- Real-time posture analytics overlay
- Clean modern UI
- Fast and lightweight performance
- Portfolio-ready AI fitness application

## Installation

### Clone Repository

```bash
git clone https://github.com/Sathvik0728/PoseAI-Pro-Posture-and-Squat-Detection.git
```

### Move Into Project Folder

```bash
cd PoseAI-Pro-Posture-and-Squat-Detection
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Project

```bash
streamlit run app.py
```
